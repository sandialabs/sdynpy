Creates an FRF matrix given ffts of responses and references

This function creates a nf x no x ni FRF matrix from the ffts provided.

Parameters

  • references (ndarray) ‚Äì A ni x nf or nf array where ni is the number of references and nt is the number of frequencies in the fft.

  • responses (ndarray) ‚Äì A no x nf or nf array where no is the number of responses and nf is the number of time frequencies in the fft.

  • method (str in ['H1','H2','Hv','Hs','LRM']) ‚Äì The method for creating the frequency response function.

  • freqs_in (ndarray) ‚Äì Only used if method == ‚ÄòLRM‚Äô. A nf or nf x 1 array of frquency values.

  • **lrm_kwargs ‚Äì Additional keyword arguments specify parameters if method == ‚ÄòLRM‚Äô. Possible arguments are: f_out (ndarray), bandwidth (float), transient (boolean), modelset (iterable of (3,1) ndarray), max_parallel (int), export_ratio (float). See sdynpy_lrm.frf_local_model.

Returns

  • H (ndarray) ‚Äì A nf x no x ni array where nf is the number of frequency lines, no is the number of outputs, and ni is the number of inputs. The output frequency lines will correspond to the frequency lines in the ffts.

  • freqs_out (None or ndarray) ‚Äì None unless method == ‚ÄòLRM‚Äô. See sdynpy_lrm.frf_local_model.

  • model_data (None dict) ‚Äì None unless method == ‚ÄòLRM‚Äô. See sdynpy_lrm.frf_local_model.

Notes

There are requirements for the shapes of references and responses for some FRF computation methods. No averaging is performed by this function.
