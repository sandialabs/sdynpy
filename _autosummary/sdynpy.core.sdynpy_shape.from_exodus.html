Reads shape data from displacements in an Exodus file

Parameters

  • x_disp (str, optional) ‚Äì String denoting the nodal variable in the exodus file from which the X-direction displacement should be read. The default is ‚ÄòDispX‚Äô. Specify None if no x_disp is to be read.
