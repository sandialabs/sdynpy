Superclass of the core SDynPy objects

The core SDynPy object arrays inherit from this class. The class is a subclass of numpy‚Äôs ndarray. The underlying data structure is stored as a structured array, but the class‚Äôs implementation allows accessing the array fields as if they were attributes.
