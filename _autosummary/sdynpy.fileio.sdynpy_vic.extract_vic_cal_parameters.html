sdynpy.fileio.sdynpy_vic.extract_vic_cal_parameters

extract_vic_cal_parameters(z3d_file)[source]

Extracts camera calibration parameters from a VIC3D .z3d file.

Parameters

z3d_file (str) – File path to the VIC3D .z3d file.

Returns

out_data – A dictionary with keys ‘intrinsics_0’, ‘extrinsics_0’, ‘distortion_0’, ‘intrinsics_1’, ‘extrinsics_1’, and ‘distortion_1’, which contain the intrinsic, extrinsic, and distortion parameters for the left camera (0) and right camera (1).

Return type

dict
